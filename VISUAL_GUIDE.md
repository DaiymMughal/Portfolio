## Interactive Elements

### 🎯 Animations You'll See:

1. **Preloader**: Spinning loader when page loads
2. **Typing Effect**: Hero title cycles through multiple titles
3. **Scroll Animations**: Elements fade in as you scroll
4. **Skill Bars**: Animate when scrolled into view
5. **Particle Background**: Interactive particles in hero
6. **Hover Effects**: 
   - Buttons scale and glow
   - Cards lift up
   - Social icons bounce
   - Project cards tilt in 3D
7. **Smooth Scrolling**: Click nav links for smooth scroll
8. **Back to Top**: Button appears when scrolling down

### 📱 Mobile View:
- Hamburger menu
- Stacked layout
- Touch-optimized buttons
- Optimized images
- Single column design
